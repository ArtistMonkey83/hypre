Parameters that are helpful for finite element information.

AConjugateProjection xxx
where xxx specifies the number of previous solution vectors to keep for the A-conjugate projection. The default is 0 (the projection is off).
minResProjection xxx
where xxx specifies the number of previous solution vectors to keep for projection. The default is 0 (the projection is off).
haveFEData
indicates that additional finite element information are available to assist in building more efficient solvers.
haveSFEI
indicates that the simplified finite element information are available to assist in building more efficient solvers.
